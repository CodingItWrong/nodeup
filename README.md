# node

Script to set up a new Node project with a few important libraries, including:

- **Testing**: [Jest][jest]
- **Code Standardization:** [ESLint][eslint] and [Prettier][prettier]

## Requirements

- Node
- [Yarn][yarn]

## Usage

Add `nodeup/bin` to your `PATH`.

```
# nodeup my-new-app-name
```
